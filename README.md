# Weather App

## Project Description

Weather App is a Python-based application that provides real-time weather information for any city. The application uses the OpenWeatherMap API to fetch weather data and displays it through a simple graphical user interface built with Tkinter.

## Features

* Search weather by city name
* Display temperature in Celsius
* Display humidity percentage
* Display weather condition
* Error message for invalid city names
* Current date and time display

## Technologies Used

* Python
* Tkinter
* Requests Library
* OpenWeatherMap API

## API Key Setup

1. Create an account on OpenWeatherMap.
2. Generate an API key.
3. Replace:
    API_KEY = "YOUR_API_KEY"
    with your own API key.

## How to Run

1. Install Python.
2. Install the required package:
   pip install requests
3. Open the project folder.
4. Run:
   python main.py
